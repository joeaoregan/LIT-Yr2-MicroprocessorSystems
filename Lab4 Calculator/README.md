<h6>Lab4 Perform calculation</h6>

### Lab 4 – Loops and offsets

In this lab you will write a program to Read in numbers using a loop and store the data at memory locations using offsets. The program will then print back all the numbers entered using a loop

In the second part of the lab you will enhance the program to either add up all the numbers entered or multiply all the numbers entered and report back the answer in decimal format. The calculation should use a loop.

1. – The program should start at address $1000
2. – Memory storage should start at $2000
3. – Use a repeat until loop to read in numbers from the keyboard until 0 is entered.  The numbers should be read in as words i.e. 16 bits The code is provided on moodle
4. – Store the numbers in memory starting at memory location $2000

5. – Use a loop to print all the numbers back to screen and demo it to your lecturer.

6. –  Save the program and call it loop-read.x68

7. – Part 2 – Add or multiply all the numbers
8. – Modify the program above to ask the user if they want to add up or multiply all the numbers.
9. – Use a decision statement to read the user selection for add or multiply and branch to the appropriate piece of code
10. – Use a loop to add or multiply the numbers 
11. – Print the answer in decimal format.
12. – Demo the program to your lecturer
13. – Save the program as loop-calc.x68
14. – Document both programs in your lab report and explain how they work. Please include an intro and conclusion.
