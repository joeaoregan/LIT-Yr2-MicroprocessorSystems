### Lab 2 – Displaying strings, Keyboard input and TRAP instructions for the 68000

In this lab you will learn how to display strings/messages and handle keyboard input in the easy68k environment. Please include an introduction and a conclusion in your lab report along with code and analysis

1. – Enter the program on page 2 of Hello World Assembly Demo and document the output and the values in D0 and A1
2. – Modify the program as documented in page 3. Document/explain the modifications made
3. – What does Trap task  14 do?
4. – Modify the program as documented in page 4. Document/explain the modifications made
5. – Modify the program as documented in page 5. 
6. – Download the planets source code from the moodle page.
7. - Modify the planets program so that for planet earth it states ‘Hello Earth, my home !’
8. – Explain how keyboard input is handled in the planets programme (hint look at Trap task 4)
